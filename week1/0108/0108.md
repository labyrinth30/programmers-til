# 웹의 이해
웹은 **월드 와이드 웹**의 줄임말이에요.
인터넷을 통해 정적인 하이퍼텍스트 문서를 주고받으며 사람들이 쉽게 정보를 검색하고 접근할 수 있도록 생겼어요.
우리는 브라우저를 통해, 하이퍼텍스트로 이루어진 웹 문서에서 다양한 콘텐츠에 접근할 수 있게 되었어요.

## 웹의 구조
크게 요청을 하는 **클라이언트(프론트엔드)**와 요청을 받아 응답을 제공하는 **서버(백엔드)**로 나뉘어요.

## 웹 개발 직무의 이해
![](https://velog.velcdn.com/images/ayeon0/post/0179e5e9-1f50-4831-8b7f-9868317bb72e/image.png)
**프론트엔드** : 웹에서 사용자쪽을 담당해서 사용자와 상호작용을 담당해요
**백엔드** : 웹의 뒷쪽에서 프론트엔드에서 전달받은 데이터와 요청을 내부 데이터와 연산을 활용해서 처리하고 프론트엔드에 결과를 전달해요
**데이터베이스** :
웹 애플리케이션이 데이터를 저장하고 관리하기 위한 시스템이에요.
서버는 데이터베이스 서버와 연결하여 사용자가 요청하는 데이터를 조회하거나 업데이트할 수 있어요.

### HTML 문법 및 태그
HTML은 'tag'가 가장 중요해요. tag란 HTML 코드에서 정보를 정의하는 형식을 뜻한다.
```<태그명 속성명1="속성값1" 속성명2="속성값2"> 컨텐츠 </태그명>```
> tag 사용시 주의점!
열리는 태그가 있으면 닫히는 태그가 있어야 해요.
닫는 태그는 태그 명 앞에는 '/'가 붙어요.
`<br />` 처럼 단독으로 쓰이는 태그도 있어요

```html
<a href="가고싶은 경로">누르면 되는 것</a>
```
을 사용하면 사용자를 이동시킬 수 있어요
```html
<input type="text" name="username">
<input type="submit" value="Login">
```
`input` 태그를 사용해서 입력을 받을 수 있어요
